Foodpanda-like PHP Starter
--------------------------
Drop this folder into your webserver root (e.g., XAMPP htdocs/foodpanda) and edit db.php.

Steps:
1. Create MySQL DB and run setup.sql.
2. Edit db.php with DB credentials.
3. Ensure uploads/ is writable.
4. Open index.php in browser.

This starter includes simple Bootstrap UI for basic flows: register, login, browse restaurants, add to cart, checkout, admin pages, and driver pages.
